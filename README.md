# NetCoreWebAPI
Bu projede, tamamen sıfırdan başlayarak Entity, DTO, DataAccess, Business ve API katmanlarını kurdum ve aralarındaki bağımlılıkları doğru şekilde yönettim.
Entity Framework Core ile veritabanı modellerini oluşturdum, ilişkileri yapılandırdım ve migration’lar ile veritabanını yönettim.
Generic Repository ve Generic Service yapıları kurarak tüm CRUD işlemlerinde tekrar eden kodları ortadan kaldırdım.
DTO yapılarını oluşturarak veritabanı modellerini API üzerinden kontrollü ve güvenli bir şekilde dışarı aktardım.
AutoMapper ile DTO–Entity dönüşümlerini yönettim.
Swagger üzerinden tüm endpoint’leri test ederek API'nin uçtan uca çalışmasını sağladım.

Bu proje, profesyonel bir .NET Core backend mimarisinin temelini oluşturan tüm yapıları deneyimlememi sağladı.
